# CJB Thread Builder

A skill for turning Due Dilly archival content into polished X thread drafts — with built-in editing and export options. Works in Claude and ChatGPT.

---

## How to Use It

### Step 1 — Get the skill

Open the file called `SKILL.md` in this repo. Click the **Raw** button. Select all the text and copy it.

### Step 2 — Open Claude or ChatGPT

Start a new conversation. Paste the skill text at the top.

### Step 3 — Get your source content

Pull any piece of Due Dilly writing — a newsletter issue, a section, a specific analysis. You can grab it from the backend or from wherever you have access to the archive. Copy the text you want to thread.

### Step 4 — Paste your content

In the same conversation, paste your Due Dilly text directly below the skill. Hit send.

### Step 5 — Review the draft

Claude or ChatGPT will return a numbered thread draft in your voice. Read through it. Check the `[verify]` flags — those are facts pulled from the source that you should confirm before posting.

### Step 6 — Edit if needed

Reply with any changes:

- "Rewrite tweet 4"
- "Sharpen the hook"
- "Cut it to 8 tweets"
- "Make tweet 6 more specific"

Go back and forth until it's right.

### Step 7 — Export

When you're happy with the thread, type one of the following:

| Command | What it does |
|---|---|
| `export notes` | Clean block to paste into Apple Notes — includes title and date |
| `export doc` | Formatted for Google Docs with dividers between tweets |
| `export post` | One tweet at a time, ready to copy and post to X |

### Step 8 — Save your draft

If you're not posting yet, copy the `export notes` output and paste it into a new Apple Note. Name it by topic. That's your thread library — come back to it anytime.

---

## Tips

- You don't need to use the whole newsletter. A single section or argument works just as well.
- The skill is built around the Due Dilly voice — analytical, specific, no fluff. The more precise your source material, the sharper the thread.
- If a draft isn't landing, try giving it a tighter angle: "focus only on the valuation section" or "make the hook about the tax structure."
