# Setting Up CJB Thread Builder as a Permanent Tool

Instead of pasting the skill every time, you can install it once and have it ready to go whenever you need it.

---

## Option 1 — Claude Projects (Recommended for Claude users)

1. Go to [claude.ai](https://claude.ai)
2. In the left sidebar, click **Projects → New Project**
3. Name it something like "CJB Thread Builder"
4. Click **Project Instructions**
5. Open `SKILL.md` in this repo, click **Raw**, select all, and copy
6. Paste it into the Project Instructions field and save

**To use it:**
Open the project, start a new conversation, and paste your source text. The skill is already loaded — no need to paste it again.

---

## Option 2 — Custom GPT (Recommended for ChatGPT users)

1. Go to [chat.openai.com](https://chat.openai.com)
2. Click **Explore GPTs** in the left sidebar
3. Click **Create** in the top right
4. In the **Instructions** field, open `SKILL.md` in this repo, click **Raw**, select all, copy, and paste it in
5. Give it a name: "CJB Thread Builder"
6. Click **Save** → set visibility to **Only me**

**To use it:**
Find it in your ChatGPT sidebar under "My GPTs." Open it, paste your source text, and go. It works exactly like a native ChatGPT tool.

---
